# 超星考试助手

一个 Tampermonkey 用户脚本，用于超星（学习通）考试整卷预览页面，提供解除限制、导出题目、批量填入答案等功能。

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 将 `chaoxing-exam-helper.user.js` 拖入 Tampermonkey 面板，或新建脚本粘贴内容
3. 保存后访问超星整卷预览页面，右上角会出现浮动控制面板

## 匹配页面

- 含"整卷预览"的页面标题
- `*.chaoxing.com/exam-ans/*`
- 本地 `file://` 打开的整卷预览页面

## 功能

### 🔓 解除限制

一键解除页面上的所有操作限制：

| 解除项 | 说明 |
|---|---|
| 文本选择 | 恢复鼠标选中文字 |
| 右键菜单 | 恢复右键功能 |
| 复制/剪切/粘贴 | 恢复剪贴板操作 |
| 键盘事件 | 移除快捷键拦截 |
| 简答题输入 | 解除 `.sub_que_div` 的 `pointer-events: none`，允许点击和输入 |
| UEditor 粘贴 | 移除富文本编辑器的 `beforepaste` 拦截，允许粘贴内容 |
| 倒计时 | 停止页面计时器 |

> 脚本加载 500ms 后自动执行一次解锁，也可点击按钮手动再次执行。

### 📤 导出题目

将当前页面的所有题目、选项和正确答案导出为文本：

```
【1. 单选题】以下哪项是...
  A. 选项A
  B. 选项B
  C. 选项C
  D. 选项D
  ✅ 答案: A (选项A)
```

弹出窗口中可以**一键复制全部**。

### ⚡ 自定义填入答案

按题号指定答案，批量填入并自动保存。支持格式：

```
1.A 2.B 3.C 4.D 5.ABC 6.对 7.这是简答题答案
```

| 题型 | 格式示例 | 说明 |
|---|---|---|
| 单选题 | `1.A` | A/B/C/D |
| 多选题 | `2.ABC` | 多个字母连写 |
| 判断题 | `3.对` 或 `3.A` | 对/是/✓/A 表示正确，错/否/✗/B 表示错误 |
| 简答/论述 | `4.这是答案内容` | 支持包含空格的文本 |
| 填空题 | `5.答案1;答案2` | 多个空用分号分隔 |

> **快捷键**：在输入框内按 `Ctrl+Enter` 快速填入。

## 原理

超星整卷预览页面的 HTML 中通过 `hidden input`（`<input id="answerXXX" value="A">`）和 CSS 类（`.check_answer`）隐藏了每道题的正确答案。脚本从 DOM 中读取答案数据，模拟点击选项、写入 hidden input、触发页面原生 AJAX 保存，同时更新整卷预览面板的题目卡片状态。

## 操作提示

- 面板可**拖拽**移动位置
- 点击 **▬** 最小化面板，点击 **✕** 关闭面板
- 状态反馈在面板底部显示，成功消息 3 秒后自动消失
- 填入答案后整卷预览面板中的题目卡片会变为蓝色（已作答状态）

## 技术细节

- 纯 JavaScript，无外部依赖
- 使用 UEditor API (`UE.getEditor().setContent()`) 设置简答题内容
- 自动调用页面原生 `submitForm` / `changeQuestionCardColor` 保持状态一致
